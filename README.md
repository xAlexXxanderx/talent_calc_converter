Convert talents between [wowhead.com/wotlk/talent-calc format](https://www.wowhead.com/wotlk/talent-calc) (also used in [WoWSims](https://wowsims.github.io/wotlk/)) and formats for export to addons (for WoW 3.3.5 version):
* Old Talented: [Talented r660](https://www.wowace.com/projects/project-2386/files/444538) / [Talented v2.4.6c-release](https://www.wowace.com/projects/project-2386/files/441991) ([MMO Champion](http://talent.mmo-champion.com) format)
* New Talented: [Talented - Kader Edition](https://github.com/bkader/Talented_WoTLK) ([Altervista](http://rpgworld.altervista.org/335/) format)

Keys:

`-s`, `--standart` or without key -- convert talents from modern Wowhead format into Talented (`MMO Champion`/`Altervista`) format

`-r`, `reverse` -- convert talents from Talented (`MMO Champion`/`Altervista`) format into modern Wowhead format
